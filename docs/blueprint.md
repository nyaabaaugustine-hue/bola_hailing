# **App Name**: BorlaHaze

## Core Features:

- GPS-based Pickup Request: Allow users to request trash pickup based on their GPS location and resolve landmark-based direction
- Waste Classification AI: Utilize AI to classify waste types based on user-uploaded images, estimating volume and weight, and categorizing it for proper disposal.
- Dynamic Pricing: Dynamically calculate the pickup price based on distance, waste type, demand, and other factors.
- Mobile Money Integration: Integrate Mobile Money payment options (MTN, AirtelTigo, Vodafone) for seamless transactions.
- Real-time Collector Matching: Match users with available collectors based on proximity, truck capacity, and waste type compatibility.
- Route Optimization Engine: Optimize pickup routes for collectors, considering traffic, road conditions, and landfill availability. This tool minimizes travel time and fuel consumption.
- USSD Booking for Feature Phones: Provide USSD-based booking for users with feature phones, enabling access without smartphones.

## Style Guidelines:

- Primary color: Vibrant orange (#FF8C00) to reflect the energy and vibrancy of local communities.
- Background color: Light gray (#F0F0F0) for a clean, uncluttered interface.
- Accent color: Forest green (#228B22) to symbolize environmental sustainability and eco-friendliness.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy feel in headings.
- Body font: 'Inter' (sans-serif) for clear readability in body text.  Recommendation: Pair 'Space Grotesk' for headlines and 'Inter' for body text
- Use simple, intuitive icons to represent waste types, pickup status, and navigation elements.
- Subtle animations to indicate loading, confirmation, and status changes for improved user experience.
